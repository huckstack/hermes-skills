# Hermes Agent Skills & Knowledge Base

Skills, worldview, and knowledge base for the Hermes Agent autonomous orchestration engine.

## Structure

- `worldview.md` — Philosophy of good agency (distilled from 2600 lessons)
- `skills/` — Individual skill files, one per skill
